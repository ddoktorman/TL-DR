TL;DR is a web application built for students to catch up on their reading. Users can select their grade, class, and book, and the application returns a page with a general summary of the book, a chapter by chapter summary, character analysis, and themes. It utilizes a Python script to scrape data from websites such as Shmoop and Sparknotes, and returns this information in an easy to read way. Simply select a book from the top, or enter your own below. Enjoy!





Note: if the tl;dr page loads and the collapsible buttons aren't clickable, it means we couldn't find that book to generate a summary for you. Sorry!
